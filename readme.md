# Readme 咯

> 这是一个以 Vue为主的 微前端项目，也是一个作者写轮子的地方。

喜欢请 Star，请勿将该项目商用。

### 安装

`npm run install-all`

### 运行

`npm run dev-all`

分别运行端口：
* 主页面 `5000`
* app1 `8085`
* app2 `8086`
* navbar `8087`

### 打包

`npm run build-all`

### 审查

`npm run audit-all`
